# DSA 210 Project Proposal: Menstrual Cycle Analysis Using Personal Clue Data

In this project, I will analyze my personal menstrual cycle data collected from the Clue application to investigate patterns between cycle phases and physical and emotional symptoms.

The dataset includes cycle dates, cycle length, period duration, bleeding intensity, and various logged symptoms such as cramps, headache, fatigue, and mood changes. I will preprocess and organize this data using Python and create additional variables such as cycle day and time until next period.

My main goal is to examine whether symptoms vary systematically across different phases of the menstrual cycle. For example, I will analyze whether symptoms such as pain or mood changes increase in the days leading up to the period or during menstruation. I will use visualizations and statistical tests to identify significant patterns.

Additionally, I will explore whether these symptoms can be used to predict the onset of the next menstrual period. For this purpose, I will build a simple classification model using symptom and cycle-related features.

This project aims to provide a data-driven understanding of personal menstrual patterns and evaluate the predictive potential of symptom tracking.
